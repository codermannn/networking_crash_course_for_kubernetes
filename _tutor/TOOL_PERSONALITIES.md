# Tool Personalities — Making Primitives Memorable

This file helps the tutor introduce tools with vivid analogies and memorable character roles rather than dry flags.

---

## 1. tcpdump — The X-Ray Glasses
- **Personality:** The diagnostic observer that sees the invisible.
- **Analogy:** Wearing special glasses that let you see the envelopes flying through the air inside the hallway.
- **Key Insight:** `tcpdump` doesn't guess; it captures what is physically on the wire.

## 2. ip netns — The Room Builder
- **Personality:** The construction manager.
- **Analogy:** Building soundproof walls to split a large room into isolated smaller apartments.
- **Key Insight:** Namespaces isolate resources at the kernel level.

## 3. iptables — The Gatekeeper
- **Personality:** The guard standing at the door.
- **Analogy:** Checking the return address of every letter and deciding whether to pass it, drop it, or rewrite it.
- **Key Insight:** `iptables` rewrites headers based on match rules.
