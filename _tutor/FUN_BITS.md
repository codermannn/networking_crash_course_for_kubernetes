# Fun Bits — Network History & Anecdotes

Use these stories and historical facts to break up heavy technical concepts and keep the student engaged.

## 1. Why Port 80 for HTTP?
In 1990, Tim Berners-Lee needed a port number for his new hypertext protocol. Port 80 was simply the next available registration slot in the IANA list. There was no grand design. Every web request on Earth now uses it because it was the next number on a paper notepad.

## 2. The Great BGP Hijacks
BGP has no security by default. In 2008, Pakistan Telecom accidentally blocked YouTube globally because global routers blindly followed their fake route. The entire internet runs on a system of mutual trust gossiped between mail clerks.

## 3. The Origin of "Ping"
Mike Muuss wrote `ping` in 1983 as a quick diagnostic tool. He named it after the sound that active sonar makes, inspired by submarine navigation. It works by sending ICMP echo requests and measuring the time for the echo reply.
